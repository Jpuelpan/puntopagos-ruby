h1. PuntoPagos

h2. Installation

In *Rails 3*, add this to your Gemfile and run the bundle command.

<pre>
	gem "puntopagos"
</pre>
In *Rails 2*, add this to your environment.rb file.

<pre>
	config.gem "puntopagos"
</pre>


Create puntopagos.yml to your config folder:

<pre>
development:
  environment:        "sandbox"
  puntopagos_key:     "YOUR-API-KEY"
  puntopagos_secret:  "YOUR-APP-SECRET"

test:
  environment:        "sandbox"
  puntopagos_key:     "YOUR-API-KEY"
  puntopagos_secret:  "YOUR-APP-SECRET"

production:
  environment:        "production"
  puntopagos_key:     "YOUR-API-KEY"
  puntopagos_secret:  "YOUR-APP-SECRET"
</pre>
